# ASSETS ORGANIZATION GUIDELINES FOR MERCADOAGRO.MX

## Directory Structure
- **assets/**  
  - **images/**  
    - All image files should be stored in this folder.  
    - **subfolder1/**  
      - (if you have categories, e.g. product categories)
    - **subfolder2/**  
  
  - **videos/**  
    - All video files should go here.  
    - **tutorials/**  
    - **ads/**  
    
  - **documents/**  
    - File formats should be .pdf, .docx, etc.
    - **guidelines/**
    - **reports/**


## Asset Naming Conventions
1. Use descriptive names that accurately reflect the content.
2. Use hyphens (-) to separate words for readability.
3. Include versioning if applicable (e.g., `image-v1.jpg`).


## File Formats
- **Images:** Use .jpg or .png for images.
- **Videos:** Use .mp4 for video formats.
- **Documents:** Use .pdf for official documents.


## Usage Guidelines
- Always compress images and videos to minimize loading time.
- Ensure that all assets are backed up regularly.


## Review and Update Process
- Asset organization guidelines should be reviewed bi-annually for optimization and updates.  
- Assign a team member responsible for the asset directory oversight.


## Conclusion
Following these guidelines will help maintain a structured and organized asset storage system for MERCADOAGRO.MX, ensuring easy access and management for all team members.
